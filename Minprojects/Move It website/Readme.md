# Simple Bootstrap Website 🌐

This is a small frontend project built using **HTML, CSS, and Bootstrap** as part of my **web development learning journey**.

The project focuses mainly on **layout design and responsiveness** using Bootstrap.  
⚠️ Some interactive features are **not functional** due to the **absence of JavaScript**.

---

## 🚀 Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  

---

## 📌 Features

- Responsive layout using Bootstrap grid system  
- Clean and simple UI design  
- Gradient backgrounds  
- Bootstrap components (buttons, sections, etc.)  
- Mobile-friendly structure  

---

## 📚 What I Learned

- Creating responsive layouts with Bootstrap  
- Using Bootstrap containers, rows, and columns  
- Styling sections with CSS and gradients  
- Structuring a webpage without JavaScript  

---

## ⚠️ Limitations

- No JavaScript used  
- Interactive elements (forms, buttons, animations) are **not functional**  
- Static frontend only
